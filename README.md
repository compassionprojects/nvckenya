# NVC Kenya Website

Developed using [gatsby](https://www.gatsbyjs.com/) and [staticcms](https://www.staticcms.org/).

## Install

```sh
npm i
cp .env.example .env.development # replace the env variables with correct values
npm run dev
```

Open localhost:8000
